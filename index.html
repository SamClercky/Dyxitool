Dyxitool

Textadaptor for dyslexics

Welcome

Welcome to the site of a beautiful project. This project is an open source. It contains a tool for dyslexic people to make reading on internet easier. I used the OpenDyslexic-font, an award winning invention made for this purpose. With this extention you can change all texts on a webpage. Also a colored background can help better reading. You can use a color as the background of the text when you hover it with the cursor. Have fun!

Customize everything

We spent a lot of time making everything easy to use. You have the ability to change everything at your preference.

Features

The extension provides at this moment only two main features:

OpenDyslexic-font

Change all fonts to OpenDyslexic

Special background

When you hover text the background will change so you can read it more easily. You can customize the color to wathever you like
